System Administration Scripts

A set of reusable Bash scripts designed to automate common Linux administration tasks, focusing on repeatability, clear output and basic operational safety.

linux automation security

Overview

The goal of this project is to reduce repetitive manual work by using scripts that can be executed consistently. Scripts include basic validation, logging and a structured approach to maintainability.

Scripts Included

  • Backup script: generates compressed backups and keeps a simple log.
  • User creation: creates users from a list (CSV) and assigns groups.
  • Health checks: checks disk usage, services and basic system status.

Example Output

[OK] Backup created: /backups/backup_2025-12-18.tar.gz
[OK] Users created: 10
[INFO] Disk usage: 41%
[OK] Services running: ssh, cron

Results

The scripts improve consistency and reduce time spent on recurring tasks. They are structured to be adapted to other lab environments with minimal changes.
